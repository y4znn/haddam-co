# Visual Constitution: Orbital Ice

## Core Palette
- **Primary Core**: `#00F0FF` (Electric Cyan). Use for primary actions, active states, and "Neon" glows.
- **The Void**: `#050B14` (Deep Navy). Use for backgrounds.
- **Glassmorphism**: `bg-slate-900/80` with `backdrop-blur-md` for cards and headers. Border: `border-white/10`.
- **Forbidden**: `orange-*`, `yellow-*`, `amber-*`. REPLACE IMMEDIATELY with Cyan or Glass variants.

## Typography
- **Headings**: Clean, tech font stack (Inter/Roboto) with `tracking-tight`.
- **Body**: Readable sans-serif.

## Components
- **Buttons (Cryo State)**:
    - Base: `bg-cyan-500 text-black font-bold`
    - Hover: `shadow-[0_0_20px_#00F0FF] scale-105 transition-all`
- **Premium Borders**:
    - "Holographic" border: `border-cyan-400/50` with a subtle pulse animation.
